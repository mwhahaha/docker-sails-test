docker-sails
------
Sets up a docker container with a test sails.js app.

Usage
------
```
docker run -p 80:1337 -t -i mwhahaha/sails-test
```

Build
-----
To build the image from this repository:
```
docker build github.com/mwhahaha/docker-sails-test
```
